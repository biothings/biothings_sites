Simple to use

Just two end-points to retrieve annotations via InChIKey or any query term.

  • Query Examples

  • Retrieve chemical/drug objects based on common IDs


    GET http://mychem.info/v1/drug/FTALBRSUTCGOEG-UHFFFAOYSA-N (InChIKey)
    GET http://mychem.info/v1/drug/DB00741 (Drugbank ID)
    GET http://mychem.info/v1/drug/CHEMBL744 (ChEMBL Compound ID)
    GET http://mychem.info/v1/drug/7LJ087RS6F (UNII Compound ID)
    GET http://mychem.info/v1/drug/CHEBI:8863 (CHEBI ID)

  • Retrieve by Drug Name


    GET http://mychem.info/v1/query?q=drugbank.name:riluzole

  • Retrieve chemicals/drugs targeting protein P24942(uniprot id)


    GET http://mychem.info/v1/query?q=drugbank.targets.uniprot:P24941

  • Retrieve chemicals/drugs having indication C0419527 (UMLS CUI)


    GET http://mychem.info/v1/query?q=drugcentral.drug_use.indication.umls_cui:C0419527

  • Batch queries? Yes, supported!


    POST http://mygene.info/v1/chem
